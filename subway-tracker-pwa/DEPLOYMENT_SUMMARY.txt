  3. Select "subway-tracker-pwa" repository

  6. Wait 30 seconds
  7. Done! 🎉

⚙️ ENVIRONMENT VARIABLE SYSTEM
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

The app checks for API key in this order:

1. Vercel Environment Variable (VITE_MTA_API_KEY)
   → Set in Vercel project settings

2. Window Config Object (window.MTA_CONFIG.apiKey)
   → Can be set in index.html for hardcoded deployments

3. LocalStorage (mta_api_key)
   → User can configure in browser

4. Fallback to Mock Data
   → Shows sample data if no API key found

To use REAL MTA data on Vercel:
→ Just set VITE_MTA_API_KEY in project settings
→ App automatically switches from mock to real data
